# Приёмочные снимки (`data/input/`)

Реальные фотографии звёздного неба для приёмки и отладки распознавателя
(см. [../../docs/data-sources.md](../../docs/data-sources.md),
[../../docs/app.md](../../docs/app.md)).

## Провенанс

- **Источник:** собственная съёмка автора проекта.
- **Правообладатель:** Igor Lazarev.
- **Формат:** Windows BMP (24-bit, 740×576). Формат BMP не несёт EXIF — скрытых
  геометок/идентификаторов устройства в файлах нет.
- **Параметры съёмки** закодированы в именах файлов (усиление `Gain`, выдержка
  `Exp`/`ms`, дата-время серии), что служит near-ground-truth; астрометрическая
  привязка (RA/Dec, FOV, поза) получается решателем и фиксируется после проверки.

## Добавление новых кадров

Любой новый реальный снимок перед коммитом:

1. вносится в реестр `docs/data-sources.md` §5 (источник, правообладатель,
   лицензия, согласие);
2. для донорских кадров — с оформленным согласием (шаблон: приложение A там же)
   и стрипом EXIF; заполненные согласия в репозиторий **не** коммитятся.
